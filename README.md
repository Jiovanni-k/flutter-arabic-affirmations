# Flutter Arabic Affirmations

A modern Flutter application that delivers a collection of Arabic positive affirmations through an intuitive and responsive user interface. The application enables users to engage with motivational statements, track repetition counts, and maintain focus on daily positive thinking practices.

## Overview

Flutter Arabic Affirmations is built with simplicity, performance, and usability in mind. The application presents affirmation cards in a clean layout, allowing users to increment repetition counters and reset progress independently for each affirmation.

The project demonstrates Flutter best practices, responsive UI design, state management fundamentals, and clean component-based architecture.

## Features

* Responsive design for mobile, tablet, and desktop screens
* Beautiful Arabic typography using Google Fonts
* Interactive affirmation cards
* Individual repetition counters
* One-click counter reset functionality
* Clean and modern Material Design interface
* Lightweight and performant implementation

## Built With

* Flutter
* Dart
* Material Design
* Google Fonts (Cairo)

## Architecture

The project follows a simple and maintainable structure:

```text
lib/
├── main.dart
├── my_home_page.dart
└── affirmation_card.dart
```

### Components

* **main.dart** – Application entry point and theme configuration.
* **my_home_page.dart** – Main screen responsible for displaying affirmation data.
* **affirmation_card.dart** – Reusable UI component representing a single affirmation card.

## Getting Started

### Prerequisites

* Flutter SDK 3.x or later
* Dart SDK
* Android Studio, VS Code, or another Flutter-compatible IDE

### Installation

```bash
git clone https://github.com/Jiovanni-k/flutter-arabic-affirmations.git
cd flutter-arabic-affirmations

flutter pub get
flutter run
```
